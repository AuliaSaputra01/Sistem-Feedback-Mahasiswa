Mengapa Menggunakan Sistem Ini?

Platform feedback mahasiswa modern berbasis Google Apps Script, Google Sheets, Bootstrap 5, Chart.js, serta dashboard admin yang mudah digunakan dan responsif di berbagai perangkat.

Google Sheets

Setiap feedback otomatis masuk ke Google Sheets tanpa perlu database tambahan.

Dashboard Statistik

Admin dapat memonitor jumlah feedback, rating dan perkembangan data secara realtime.

Export Laporan

Seluruh data dapat diunduh menjadi PDF maupun Excel hanya dalam satu klik.

Admin Login

Pengelolaan data dilakukan melalui dashboard admin yang aman dan mudah.

Responsive

Tampilan optimal di smartphone, tablet maupun desktop.

Dark Mode

Mendukung mode terang dan gelap sehingga nyaman digunakan kapan saja.
